# Scoring Model v1 (0–5)

Оценка каждой идеи по 8 критериям. Балл: 0 (плохо) → 5 (отлично).

## Критерии и веса
1. **Pain Intensity** — 0.20
2. **Willingness To Pay** — 0.20
3. **Frequency/Recurrence** — 0.15
4. **Distribution Access** (насколько быстро достаём клиентов) — 0.15
5. **Build Feasibility** (MVP за 2–4 недели) — 0.10
6. **Competition Advantage** (чем реально отличаемся) — 0.10
7. **Retention Potential** (есть ли причина остаться) — 0.05
8. **Legal/Operational Risk** (инвертированный: меньше риск = выше балл) — 0.05

## Формула
**Total Score = Σ(score_i × weight_i) × 20**

Итог: 0–100.

## Пороговые решения
- **80–100:** Build now
- **65–79:** Validate hard (ещё 7 дней)
- **50–64:** Hold
- **<50:** Kill

## Жёсткие стоп-факторы
- Нет подтверждённой боли у ICP
- Нет канала первых клиентов
- Нет понятной монетизации
